# Full-Stack-Extension-Pack

## Visual Studio 2019 Extensions Collection

This extension is useful for full stack developers.

It consists of the following extensions:

  1. Visual Studio Spell Checker (VS2017 and Later)
  1. CodeMaid
  1. Solution Error Visualizer
  1. Clear MEF Component Cache
  1. File Differ
  1. EF Core Power Tools
  1. ILSpy
  1. Clear Recent
  1. IndentRainbow
  1. Fine Code Coverage
  1. SteroidsVS
  1. AsyncFixer for Visual Studio 2019
  1. Experimental Tools
  1. Open Command Line
  1. Markdown Editor
  1. Run Coverlet Report
  1. Viasfora
  1. Add New File
  1. Editor Enhancements
  1. File Icons
  1. Web Compiler
  1. ZenCoding
